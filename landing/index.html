The CDN platform for students, nonprofits, amateur radio operators, and the open source community.

delivr.dev is an open source CDN for authoritative DNS and HTTP mirroring.

Learn more

The delivr.dev CDN

The delivr.dev CDN is powered by AS34553 and backed by a global network of CDN PoPs for students, nonprofits, and the amateur radio community.

Fully Open Source

All the code behind delivr.dev is open source. From the delivr.dev code itself and all the third party components that power the platform including MongoDB and Ansible, the entire software stack is open source.
